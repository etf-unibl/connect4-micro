Detailed Description

This source file contains functions for abstracting connectfour game. It's possible to initialize connect four game, insert disc in appropriate column, switch between red and green player, check if one of players wins the game, turn on or off winners discs and start new game.

◆ ConnectFour_CheckWinner()

unsigned char ConnectFour_CheckWinner ( )

Checks for winner.

The function cheks if current player has four disc connected together. The discs can be connected by x or y axis, or by both of diagonales.

Returns
Returns 1 if somone wins the game, or 0 otherwise
Note
Reads mGameMatrix, mCurrentPlayerColor and modify mWinnersDiscsPosition

unsigned char ConnectFour_InsertDisc ( unsigned char  column)

Inserts disc into the appropriate column.

The function iserts the disc into mGameMatrix, then simulates insertion of the disc into the display. Finnaly, it checks if the disc is inserted successfully.

Parameters
columnDisc column
Returns
Returns 1 if disc inserted successfully, 0 otherwise
Note
Reads mCurrentPlayerColor and modify mGameMatrix, mDiscXPosition, mDiscYPosition
Warning
Parameter column must be between 1 and 7
